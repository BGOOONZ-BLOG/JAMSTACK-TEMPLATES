# Notes from EmberConf 2015 Talks

My collection of notes from [EmberConf 2015](http://emberconf.com/).

## Installation

```
git clone https://github.com/johnotander/emberconf2015-notes.git
cd emberconf2015-notes
jekyll serve --watch
```
