# Unstable_A11y

Experimental utility for validating accessibility at runtime. See the example below for mvp
functionality.

It uses the [axe-core](https://github.com/dequelabs/axe-core) library to perform the validations.
[Here](https://dequeuniversity.com/rules/axe/3.2) is a link to the latest rules with descriptions
and suggestions to fix.

Please give this component a try to see if it helps in your application, and we welcome any
suggestions for how to improve.
