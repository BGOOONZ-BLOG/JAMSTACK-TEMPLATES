# Application navigation bar

<Yard placeholderHeight={134} {...appNavBarYardConfig} />

The navigation bar standardizes and creates a shared navigation experience across the applications and products.
It enables users to orient and navigate different sections of applications reliably.

You may need to render some icon or other non-text element in the menu item and the `mapItemToNode` prop is how
to do it. Follow the example below to see how it renders colored borders on the item.
