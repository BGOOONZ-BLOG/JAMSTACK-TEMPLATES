Bleuprint

Yet another scaffold for static webpages.

Prototype quickly.

Don't waste your time writing the same old boilerplate. Just clone and modify it to fit your own needs. Bleuprint is intended to get you up and running quickly with a static website. It's quite useful for building a product idea splash page to measure market interest.

Mobile-first design

Responsive out of the box, ensuring a consistent and cohesive experience for all devices.

SCSS/OOCSS

It's easy to theme and adjust layout by simply adjusting variables and adding classes to HTML elements.

Gulp integration

Automated build tasks for SCSS compilation and minification, Javascript uglification, and error checking.

The defaults are sane. Seriously.

It's built with BASSCSS, clrs, and Jekyll.
