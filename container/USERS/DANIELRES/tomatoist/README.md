Dependencies
------------

* [rubygems](http://rubygems.org)
* [bundler](http://gembundler.com)
* [mongodb](http://mongodb.org)

Running the App
---------------

1. Install the gems:
  <code><pre>
    gem install bundler
    bundle install
  </pre></code>
2. Run the server
    <code><pre>padrino start</pre></code>
3. [stop! pomodoro time!](http://localhost:3000)
