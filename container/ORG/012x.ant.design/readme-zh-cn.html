Ant Design

安装

bash npm install antd

示例

jsx import { DatePicker } from 'antd'; ReactDOM.render(<DatePicker />, mountNode);

引入样式：

jsx import 'antd/lib/index.css'; // or 'antd/style/index.less'

按需加载可通过此写法 import DatePicker from 'antd/lib/date-picker' 或使用插件 babel-plugin-antd。

浏览器支持

现代浏览器和 IE8 及以上。
