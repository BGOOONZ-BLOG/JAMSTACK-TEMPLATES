# Skeleton

<Yard placeholderHeight={44} {...skeletonYardConfig} />

The `Skeleton` component is optimized for rendering shapes with or without animation while waiting for loading corresponding components.

## Examples

<Example title="Basic usage" path="skeleton/basic.js">
  <SkeletonBasic />
</Example>

Set `height` and `width` to generate a single rectangular `Skeleton`.

<Example title="Other Shapes" path="skeleton/circle.js">
  <SkeletonCircle />
</Example>

You can change the shape and position of `Skeleton` through `overrides.Root`.

<Example title="Animation" path="skeleton/animation.js">
  <SkeletonAnimation />
</Example>

<Example title="Sub elements" path="skeleton/subElements.js">
  <SkeletonSubElements />
</Example>

Set `rows` to a positive number and `Skeleton` will generate multiple sub elements with the same shape inside the `Root`.
The background color of `Root` will become transparent automatically when the `rows` equals to a positive number.
You can customize the sub elements' shape and position through `overrides.Row`.
